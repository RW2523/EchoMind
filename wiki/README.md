# Wiki pages (staged)

These are the GitHub Wiki pages, kept in the repo so they're versioned and reviewable.

- `Home.md`, `User-Guide.md`, `Architecture.md`, `FAQ.md`, `_Sidebar.md`

## Publishing

The GitHub Wiki is currently **unavailable** for this repo: wikis on **private**
repositories require a paid GitHub plan (Pro/Team). Options:

1. Make the repo **public**, or upgrade to **GitHub Pro/Team**, then enable
   *Settings ▸ Features ▸ Wikis*.
2. Create the first wiki page once in the web UI (this initialises the wiki git repo).
3. Run `./wiki/publish_wiki.sh` to push all pages.

Until then, the identical content is browsable at [docs/](../docs/).
